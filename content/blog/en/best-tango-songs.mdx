## Listening Suggestions

### 1. Orchestra-Based Listening

Listen only to D'Arienzo for the first week. Learn its character. Then move to Di Sarli. Get to know each orchestra separately.

### 2. Genre-Based Listening

One day only tangos, one day only milongas, one day only vals.

### 3. Active Listening

- Keep the rhythm while music plays
- Count strong beats
- Hum the melody
- Imagine how you would dance

## Finding on Spotify

You can find most of these songs on Spotify. When searching:

```
"[Song Name]" "[Orchestra Name]" tango
```

Example: `"La Cumparsita" "D'Arienzo" tango`

<Callout type="info" title="Note">
There are versions of the same song played by different orchestras. Don't forget to write the orchestra name.
</Callout>
